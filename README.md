# Raycast

## Sobre o Projeto

Raycast é um jogo desenvolvido na Unity utilizando a linguagem C#. O objetivo do jogador é sobreviver até que o cronômetro chegue a zero, utilizando diferentes tipos de Raycast para eliminar os inimigos que surgem durante a partida.

O jogo possui três categorias de inimigos, cada uma com características próprias e formas específicas de eliminação. Durante a partida, o jogador deve administrar sua vida, reagir rapidamente aos inimigos e utilizar corretamente cada sistema de Raycast disponível.

Além disso, a dificuldade aumenta progressivamente conforme o tempo avança, tornando a experiência mais desafiadora e exigindo maior atenção do jogador.

## Principais Mecânicas

* Movimentação horizontal do jogador.
* Sistema de vida.
* Cronômetro regressivo.
* Três tipos de Raycast.
* Três tipos de inimigos.
* Sistema de dificuldade progressiva.
* Telas de vitória e derrota.

## Tecnologias Utilizadas

* Unity
* C#
* TextMeshPro
